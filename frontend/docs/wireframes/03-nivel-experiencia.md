# Wireframe: Nivel de Experiencia

## Elements:
- **Navigation/Header:**
  - Back button.
  - Progress bar (Step 2 of X).

- **Main Content Area:**
  - Title: "Select Your Experience Level"
  - Options for experience levels:
    - Essenza
    - Explora
    - Explora+
    - Bivouac
    - Atelier
  - Each option should have a brief description of what it includes.
  - Visual indicator for selected level.
  - Note about "Solo Traveller" +50% price markup (if applicable, based on previous selection).

- **Call to Action:**
  - Button: "Continue to Basic Configuration"

## Layout:
- Options presented as cards or selectable buttons.
- Clear visual feedback on selection.

## Interactions:
- Selecting an experience level updates the potential price (if Solo Traveller).
- Clicking "Continue" navigates to the "Configuración Básica" step.
